# Interactive Video Gallery with Reusable React Components
  This project is a modern, responsive, and interactive video gallery built with React and Tailwind CSS. It showcases reusable components such as Card, Button, and Modal across multiple pages including Movie, Drama, Cartoon, and Video pages.

# Each Card can display video previews, images, or styled placeholders, and comes with a Button    to trigger a Modal for detailed viewing. The gallery supports hover effects, smooth transitions, and responsive grid layouts, making it visually appealing on all devices.

#Features include:

-> Autoplaying, muted, looped video previews inside cards.
-> Modal component for full video or content display.
-> Tailwind CSS for animations, hover effects, and responsive design.
-> Reusable components for easy scalability and maintainability.
-> Easy integration with multiple pages and dynamic content.
-> This project is ideal for showcasing portfolios, movie galleries, or interactive media -      applications, and demonstrates best practices in React component design, responsive layouts, and  user interaction.

Explore more: Click on any card to view videos or details and experience smooth transitions and animations in a modern web interface.
